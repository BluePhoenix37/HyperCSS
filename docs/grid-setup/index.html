Grid Setup

60 Columns

The Hyper grid system has 60 columns. This allows you to have 1 grids, 2 grids, 3 grids, 4 grids, 5 grids, 6 grids, 10 grids, 12 grids, and more. The syntax is pretty easy.

Syntax

The grid system uses 4 sizes: xs (phones), sm (tablets), md (pc's), and lg (desktop large screens). Using these, you can choose how the grid works on all devices.


Since the grid has 60 columns, make sure your columns for each row add up to 60. To add a grid, add each grid using the grid class for the container. Then, each piece of the grid has a class grid-<device>-<length> So let's take our HTML document, and try this out.

<html>

     <head>

         <title>My First HyperCSS Document</title>

         <link rel="stylesheet" href="https://hypercss.glitch.me/archives/hyper.css">

     </head>

     <body class="t-sans-serif">

       <div class="grid">

         <div class="t-hg grid-md-20 grid-lg-20 grid-sm-30 grid-xs-30">This is my first HyperCSS document</div>

         <div class="t-ms grid-md-40 grid-lg-40 grid-sm-30 grid-xs-30">Isn't that cool?</div>

     </body>

</html>

Other things to do with the grid

If you want the same grid for all device sides, and you don't want to have to type out everything, simply use grid-length without the device inbetween, and that will stay the same grid for every device;

Also, if you want to hide or show certain items on devices, use <device>-hidden or <device>-show.
