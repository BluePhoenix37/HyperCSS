Getting Started

Choosing your version

HyperCSS has two versions. The full version, and the pure grid version. The full version has more classes and more styling capabilities, while the pure grid version has just the responsive grid.

Adding Hyper to your project

To add the full version, simply add this code snippet inside the <head> tag:


<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/BluePhoenix37/HyperCSS@main/archives/hyper.min.css">

To add the pure grid version, add this snippet instead:


<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/BluePhoenix37/HyperCSS@main/archives/grid.min.css">

Now that we know what style we are choosing, let's create a webpage.

<html>

     <head>

         <title>My First HyperCSS Document</title>

         <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/BluePhoenix37/HyperCSS@main/archives/hyper.min.css">

     </head>

     <body>

         <h1>This is my first HyperCSS document</h1>

         <p>Isn't that cool?</p>

     </body>

</html>
