Classes and Terms

The Two Types

Hyper three main types of classes. Size, color, and word. Each follows the same patterns, so learning these is the first step on learning Hyper.

Size

Size tags have 7 "options". For example, take text size. The 7 options are as follows:


  1. t-sm (smallest)

  2. t-ms (medium-small)

  3. t-md (medium)

  4. t-ml (medium-large)

  5. t-lg (large)

  6. t-hg (huge)

  7. t-en (enourmous)

So when changing text size, you would use t-<size> So let's take our HTML document, and try this out.

<html>

     <head>

         <title>My First HyperCSS Document</title>

         <link rel="stylesheet" href="https://hypercss.glitch.me/archives/hyper.css">

     </head>

     <body>

         <h1 class="t-en">This is my first HyperCSS document</h1>

         <p class="t-ms">Isn't that cool?</p>

     </body>

</html>

The other size classes are bw-size (border-width), rd-size (border-radius), and p-sm (padding)

Color

Hyper has four color schemes, with more coming. The four color schemes are red, blue, green and grey. To change the darkness, add numbers on the end. red-100 is red, and red-50 is maroon. (The numbers go by tens, so red-10 is the darkest, red-20 is second darkest and so on and so on)

For changing text-color, use t-<color>-<amount>

<html>

     <head>

         <title>My First HyperCSS Document</title>

         <link rel="stylesheet" href="https://hypercss.glitch.me/archives/hyper.css">

     </head>

     <body>

         <h1 class="t-en t-red-100">This is my first HyperCSS document</h1>

         <p class="t-ms t-blue-100">Isn't that cool?</p>

     </body>

</html>

Some other color elements include bd-color (border) and b-color (background-color)

Note: A feature that all color classes have are selectors. If you want to change color on hover or focus, add hover- or focus- before, like hover-b-blue-90

Word

Word classes are the misc. section. Some examples include t-sans-serif, t-break and pos-relative
