Customization

config.css

The HyperCSS code already has customization built into it, using css variables. All you need to do is create and link to in your html, a config.css file.

What the file should look like

Copy this file, and add it to your project:

:root {

  /*Color*/

  --c-b-1: ;

  --c-b-2: ;

  --c-b-3: ;

  --c-b-4: ;

  --c-b-5: ;

  --c-t-1: ;

  --c-t-2: ;

  --c-t-3: ;

  --c-t-4: ;

  --c-t-5: ;

  /*Text-Size*/

  --c-ts-1: ;

  --c-ts-2: ;

  --c-ts-3: ;

  --c-ts-4: ;

  --c-ts-5: ;

  /*Border*/

  --c-bd-1: ;

  --c-bd-2: ;

  --c-bd-3: ;

  --c-bd-4: ;

  --c-bd-5: ;

  --c-bw-1: ;

  --c-bw-2: ;

  --c-bw-3: ;

  --c-bw-4: ;

  --c-bw-5: ;

  --c-rd-1: ;

  --c-rd-2: ;

  --c-rd-3: ;

  --c-rd-4: ;

  --c-rd-5: ;

  --c-grid-gap: ;

}

Then, you can fill in your custom wants. Do you want a specific hex code for background color? Put it in after the colon after the "--c-b-1" (which stands for custom background one). Then, call it using the class c-b-1.

Example:

:root {

  /*Color*/

  --c-b-1: #0096fd;

  --c-b-2: ;

  --c-b-3: ;

  --c-b-4: ;

  --c-b-5: ;

  --c-t-1: ;

  --c-t-2: ;

  --c-t-3: ;

  --c-t-4: ;

  --c-t-5: ;

  /*Text-Size*/

  --c-ts-1: ;

  --c-ts-2: ;

  --c-ts-3: ;

  --c-ts-4: ;

  --c-ts-5: ;

  /*Border*/

  --c-bd-1: ;

  --c-bd-2: ;

  --c-bd-3: ;

  --c-bd-4: ;

  --c-bd-5: ;

  --c-bw-1: 1px 0px 0px 0px;

  --c-bw-2: 0px 0px 1px 0px;

  --c-bw-3: ;

  --c-bw-4: ;

  --c-bw-5: ;

  --c-rd-1: 25%;

  --c-rd-2: 10%;

  --c-rd-3: ;

  --c-rd-4: ;

  --c-rd-5: ;

  --c-grid-gap: 2px;

}
